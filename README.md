# Post Management Platform

A post management platform built with Node.js and Express.

## Features
- User Registration and Login
- JWT Authentication
- Password hashing using bcrypt
- Create and manage posts
- Like posts

## Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT
- bcrypt

## Installation

1. Clone the repository
git clone https://github.com/yourusername/post-platform.git

2. Install dependencies
npm install

3. Run the server
node index.js
